# Design System (Gumroad-inspired, Original)

## Principles
- Minimal UI with high readability and strong hierarchy
- Neutral foundation + one accent color
- Crisp borders, restrained shadows, and generous spacing
- Fast interactions, low motion, semantic structure

## Tokens

### Color
- `--surface`: `#fcfcfc`
- `--ink`: `#111111`
- `--muted`: `#62636b`
- `--border`: `#dfdfe4`
- `--accent`: `#1f6feb`

### Typography
- Display: `Manrope`
- Body: `Work Sans`

### Radius and shadow
- radius: `8/12/16px`
- shadow: 1px crisp edge shadow only

## Layout
- Mobile-first
- Base spacing: 4px scale
- Maximum content width: 6xl

## Components (Step 2)
- Surface card
- Minimal header nav
- Section heading blocks
